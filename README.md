# Grocery Category Classification

This project classifies grocery product categories into:
- Food
- Drinks
- Home Care
- Personal Care
- Other

---

## 🔍 Problem

Given short category names (e.g., "NB Fresh Fruits"), classify them into one of the predefined groups.

---

## ⚙️ Approach

- Sentence Transformers (MiniLM embeddings)
- Logistic Regression classifier
- Context-based input enhancement

---

## 📊 Results

- Accuracy: ~85%
- Evaluated using:
  - Train-test split
  - Confusion matrix
  - Classification report

---

## 🚀 How to Run

Install dependencies:
